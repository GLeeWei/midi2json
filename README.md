# midi2json
This program converts quantized midi files to text files with PT2 note notation.
# How to install
`git clone https://github.com/Volian0/midi2json`  
`cd midi2json`  
`g++ src/*.cpp src/midi/*.cpp -std=c++11 -o midi2json`  
# How to update
`cd midi2json`  
`git pull`  
`g++ src/*.cpp src/midi/*.cpp -std=c++11 -o midi2json`
