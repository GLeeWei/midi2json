# midi2json
This program converts quantized midi files to text files with PT2 note notation.
# How to install
## Linux
`git clone https://github.com/Volian0/midi2json`  
`cd midi2json`  
`chmod +x compile.sh`  
`./compile.sh`  
## MinGW Distro
`git clone https://github.com/Volian0/midi2json`  
`cd midi2json`  
`compile.sh`  
# How to update
`cd midi2json`  
`git pull origin master`  
`./compile.sh` (Linux)  
`compile.sh` (MinGW Distro)
